CRITERIOS DE REGISTRO

Definiciones operativas

Estas reglas permiten comparar los reportes entre servicios, regiones y fechas.

Sin afectación

La residencia continúa operativa y no registra daños, interrupciones ni necesidades extraordinarias asociadas a la emergencia.

Con afectación

Existe daño, interrupción de servicios básicos, riesgo para la operación, evacuación u otra condición que requiere seguimiento.

Sin cambios

El estado, las situaciones presentes y las necesidades se mantienen respecto del reporte anterior. El envío actualiza la fecha y los datos de contacto.

Electrodependencia

Condición en que una persona requiere suministro eléctrico continuo para el funcionamiento de un dispositivo médico indispensable.

Último reporte vigente

El tablero utiliza el envío más reciente para cada combinación de servicio, región, comuna y residencia.

Protección de datos

No se deben registrar nombres, RUN, diagnósticos ni otros antecedentes personales de las personas residentes.
